# ข้อมูลจำเพาะการจัดซื้อ TUC และระบบฐานความรู้ - คู่มือการใช้งาน

ยินดีต้อนรับสู่ระบบจัดซื้ออัจฉริยะของ TUC แพลตฟอร์มนี้ผสานรวมเทคโนโลยี Gemini AI เพื่อช่วยให้คุณแยกวิเคราะห์ข้อกำหนดของอุปกรณ์ สร้างมาตรฐานทางเทคนิค และแปลหลายภาษาได้อย่างรวดเร็ว

## 1. คุณสมบัติหลัก
- **การแยกวิเคราะห์อัจฉริยะ**: แยกพารามิเตอร์ทางเทคนิคจากไฟล์ PDF/Docx โดยอัตโนมัติ
- **ฐานความรู้บนคลาวด์**: แบ่งปันการบำรุงรักษาอุปกรณ์และมาตรฐานทางเทคนิคข้ามภูมิภาค
- **การทำเหมืองข้อมูลขั้นสุด (Extreme Mining)**: สแกนเอกสารอย่างลึกซึ้งเพื่อดึงค่าทางเทคนิคทุกชิ้นออกมา
- **การซิงค์หลายภาษา**: สลับระหว่างจีนตัวเต็ม, จีนตัวย่อ, อังกฤษ และไทยได้แบบเรียลไทม์

## 2. ข้อมูลสำคัญเกี่ยวกับ Gemini API Key (BYOK)
เพื่อให้แน่ใจว่าประสิทธิภาพการแยกวิเคราะห์สูงและลดต้นทุนการดำเนินงาน ระบบนี้จึงใช้โมเดล **BYOK (Bring Your Own Key)**

### ทำไมถึงแนะนำให้ใช้ระดับฟรี (Free Tier)?
- **ไม่มีค่าใช้จ่าย**: Google AI Studio มีระดับการใช้งานฟรีที่เหมาะสำหรับการใช้งานส่วนบุคคลหรือทีมขนาดเล็ก โดยไม่มีค่าธรรมเนียม API
- **ประสิทธิภาพสูง**: แม้จะอยู่ในระดับฟรี ความเร็วและความแม่นยำในการแยกวิเคราะห์ยังคงยอดเยี่ยม
- **การควบคุมความเป็นส่วนตัว**: คีย์ของคุณจะถูกจัดเก็บไว้ในเบราว์เซอร์ภายในเครื่องของคุณเท่านั้น ระบบจะไม่ส่งต่อคีย์ส่วนตัวของคุณ

### ข้อจำกัดและกฎระเบียบ
- **ขีดจำกัดอัตรา (Rate Limit)**: ระดับฟรีมักจำกัดอยู่ที่ 15 คำขอต่อนาที (15 RPM) ซึ่งเพียงพอสำหรับการใช้งานเพียงคนเดียว
- **การใช้ข้อมูล**: โปรดทราบว่าข้อมูลที่ป้อนในระดับฟรีอาจถูก Google นำไปใช้เพื่อปรับปรุงโมเดล (โปรดอย่าอัปโหลดเอกสารที่มีความลับสูง)

### วิธีการสมัคร?
1. ไปที่ [Google AI Studio](https://aistudio.google.com/app/apikey)
2. เข้าสู่ระบบด้วยบัญชี Google ของคุณ
3. คลิก **"Create API key"**
4. คัดลอกคีย์และวางลงในช่อง API Key ในไอคอน "การตั้งค่า" ที่ด้านบนขวาของระบบนี้

## 3. ขั้นตอนการใช้งาน
1. **ตั้งค่าคีย์**: ตรวจสอบให้แน่ใจว่าคุณใส่ Gemini API Key ก่อนการใช้งานครั้งแรก
2. **อัปโหลดไฟล์**: คลิก "อัปโหลดข้อมูลจำเพาะใหม่" ระบบจะจัดคิวเพื่อแยกวิเคราะห์เบื้องหลัง
3. **โหลดจากคลาวด์**: คลิก "โหลดจากคลาวด์" เพื่ออ้างอิงบันทึกประวัติพร้อมการแปลอัตโนมัติตามภาษาของคุณ
4. **การทำเหมืองข้อมูลขั้นสุด**: หากการแยกวิเคราะห์ไม่สมบูรณ์ ให้ใช้ฟังก์ชัน "แยกวิเคราะห์ใหม่" เพื่อเรียกใช้กลไกการทำเหมืองข้อมูลขั้นสุด
