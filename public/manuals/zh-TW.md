# TUC 採購規格與知識庫系統 - 操作說明書

歡迎使用 TUC 智能採購規格系統。本系統整合了 Gemini AI 技術，協助您快速解析設備規範、建立技術標準並進行多語系轉譯。

## 1. 核心功能
- **智能解析**：自動提取 PDF/Docx 檔案中的技術參數。
- **雲端知識庫**：跨區域共享設備保養與技術標準。
- **極致挖掘 (Extreme Mining)**：深度掃描文件，榨乾每一條技術價值。
- **多語系同步**：支援繁中、簡中、英文、泰文即時切換。

## 2. Gemini API Key 重要說明 (BYOK)
為了確保系統的解析能力並降低運作成本，本系統採用 **BYOK (Bring Your Own Key)** 模式。

### 為什麼推薦使用 Free Tier (免付費等級)？
- **零成本**：Google AI Studio 提供免費額度，適合個人或小組日常使用，完全無需支付 API 費用。
- **高性能**：即便在免付費等級，解析速度與準確度依然非常出色。
- **隱私控制**：您的 Key 僅儲存於本地瀏覽器，系統不會傳輸您的私鑰。

### 限制與規範
- **速率限制 (Rate Limit)**：免付費等級通常限制在每分鐘 15 次請求 (15 RPM)，對於單人操作已非常足夠。
- **資料使用**：請注意免付費等級的輸入內容可能被 Google 用於模型優化（請勿上傳極機密文件）。

### 如何申請？
1. 前往 [Google AI Studio](https://aistudio.google.com/app/apikey)。
2. 登入您的 Google 帳號。
3. 點擊 **"Create API key"**。
4. 複製金鑰並貼回本系統右上方「設定」圖示中的 API Key 欄位。

## 3. 操作流程
1. **設定金鑰**：初次使用請務必填入申請到的 Gemini API Key。
2. **上傳檔案**：點擊「上傳新規格」，系統將自動推送到背景佇列解析。
3. **雲端載入**：點擊「從雲端載入」，可快速引用歷史記錄，系統會自動根據您的語系進行轉譯。
4. **極致挖掘**：若發現文件解析不全，可使用「重新解析」功能啟動極致挖掘機制。
