# TUC 采购规格與知识库系统 - 操作说明书

欢迎使用 TUC 智能采购规格系统。本系统整合了 Gemini AI 技术，协助您快速解析设备规范、建立技术标准并进行多语系转译。

## 1. 核心功能
- **智能解析**：自动提取 PDF/Docx 文件中的技术参数。
- **云端知识库**：跨区域共享设备保养与技术标准。
- **极致挖掘 (Extreme Mining)**：深度扫描文件，榨干每一条技术价值。
- **多语系同步**：支持繁中、简中、英文、泰文即时切换。

## 2. Gemini API Key 重要说明 (BYOK)
为了确保系统的解析能力并降低运作成本，本系统采用 **BYOK (Bring Your Own Key)** 模式。

### 为什么推荐使用 Free Tier (免付费等级)？
- **零成本**：Google AI Studio 提供免费额度，适合个人或小组日常使用，完全无需支付 API 费用。
- **高性能**：即便在免付费等级，解析速度与准确度依然非常出色。
- **隐私控制**：您的 Key 仅储存在本地浏览器，系统不会传输您的私钥。

### 限制与规范
- **速率限制 (Rate Limit)**：免付费等级通常限制在每分钟 15 次请求 (15 RPM)，对于单人操作已非常足够。
- **资料使用**：请注意免付费等级的输入内容可能被 Google 用于模型优化（请勿上传极机密文件）。

### 如何申请？
1. 前往 [Google AI Studio](https://aistudio.google.com/app/apikey)。
2. 登录您的 Google 账号。
3. 点击 **"Create API key"**。
4. 复制密钥并贴回本系统右上方「设置」图标中的 API Key 栏位。

## 3. 操作流程
1. **设置密钥**：初次使用请务必填入申请到的 Gemini API Key。
2. **上传文件**：点击「上传新规格」，系统将自动推送到背景队列解析。
3. **云端载入**：点击「从云端载入」，可快速引用历史记录，系统会自动根据您的语系进行转译。
4. **极致挖掘**：若发现文件解析不全，可使用「重新解析」功能启动极致挖掘机制。
